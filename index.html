대표 서비스

SKCT 온라인 연습 도구

사용자가 가진 문제 자료를 실제 온라인 검사 흐름에 맞춰 연습할 수 있도록 만든 웹 도구입니다. OMR, 과목별 타이머, 메모장, 그림판, 계산기, 채점, 문항별 통계를 한 화면에서 제공합니다.

  • 일반 모드: OMR·타이머·메모장·계산기 무료 제공
  • 고급 모드: 실제에 가까운 폭과 우측 여백, 팝업 연습 배치
  • 복기 기능: 정답 입력, CBT 정답률, 과목별 통계, TXT/CSV 저장
  • 기록 보관함: 회차 기록, 성장 그래프, 오답노트, 복기 메모 저장
SK그룹 또는 SKCT 공식 운영기관과 제휴된 서비스가 아닌 개인 제작 연습 도구입니다. 공식 문제나 기출 문제를 제공하지 않습니다.

운영 기준

콘텐츠 기준

AgenticFabWorks의 문서와 도구 설명은 직접 작성한 안내입니다. 각 도구는 사용자가 목적을 빠르게 달성하도록 설계하며, 기능이 불안정하거나 안내가 부족한 화면은 정식 기능처럼 홍보하지 않습니다.

광고 기준

광고는 소개, 가이드, FAQ처럼 읽는 페이지 위주로 제한합니다. 일반 모드 쉬는 시간 배너는 승인 뒤 정책을 지키는 범위에서만 검토하고, SKCT 고급 모드, 팝업 풀이창, 채팅, 보관함처럼 집중 조작이나 개인 데이터가 있는 화면에는 광고를 넣지 않습니다.
