시행일: 2026년 7월 24일. AgenticFabWorks는 기능 제공에 필요한 범위에서 브라우저 저장소, 방문 통계, 문의·신청 기록을 다룹니다. 아래 내용과 개별 서비스의 안내가 다르면 더 구체적인 개별 안내를 우선합니다.

서비스별 범위

모든 하위 도메인이 같은 방식으로 처리하지 않습니다.

대표 사이트

agenticfabworks.com의 첫 화면에는 이용 흐름을 파악하기 위한 Google Analytics가 사용됩니다. 일부 충분한 설명이 있는 문서 페이지에는 Google AdSense 광고 코드가 포함될 수 있습니다.

웹 계산 도구

tools.agenticfabworks.com에는 Google Analytics와 AdSense 코드를 두지 않습니다. 입력값과 계산 결과는 일반적으로 현재 브라우저에서 처리합니다. 디데이 목록은 사용자가 저장할 때 브라우저의 localStorage에 남습니다.

환율 계산의 예외

환율 계산기는 최신 기준값을 받기 위해 Frankfurter API에 통화 코드 목록을 요청합니다. 사용자가 입력한 금액 자체는 요청 주소에 포함하지 않습니다. API가 응답하지 않으면 화면에 기준일과 대체값 사용 여부를 표시합니다.

SKCT 연습 도구

skct.agenticfabworks.com에는 광고 코드를 두지 않습니다. 앱 첫 화면에는 페이지·화면 모드, 연습 시간 설정과 정답·응답 문항의 합계처럼 이용 흐름을 확인하기 위한 Google Analytics가 사용됩니다. 활용 도구 열기 이벤트에는 선택한 서비스 식별자만 포함하며 답안, 이메일, 메모나 검색어는 포함하지 않습니다. 기본 문제 풀이 설정과 화면 선택은 브라우저 저장소에 남을 수 있습니다. 고급 기능을 직접 신청하거나 로그인해 보관함을 사용할 때에는 신청 정보, 라이선스 상태와 학습 기록이 Firebase 기반 기능을 통해 처리될 수 있습니다.

처리 원칙

목적, 저장 위치와 삭제 방법

처리할 수 있는 항목

브라우저와 기기 정보, 접속 시각과 페이지 이용 통계, 사용자가 보낸 이메일, 고급 기능 신청 항목, 로그인 식별자, 사용자가 저장을 선택한 연습 기록을 기능별로 처리할 수 있습니다. 계산기 입력값을 별도의 회원 프로필로 수집하지 않습니다.

이용 목적

페이지와 계산 기능 제공, 사용자 설정 복원, 오류·보안 문제 확인, 고급 기능 승인과 라이선스 확인, 기록 보관, 문의 응답에 사용합니다. 광고나 분석 목적의 처리는 해당 코드가 실제로 포함된 페이지에 한정됩니다.

브라우저 저장소

localStorage와 sessionStorage에는 화면 모드, 저장한 날짜, 임시 신청 상태와 같은 값이 남을 수 있습니다. 브라우저의 사이트 데이터 삭제 기능으로 지울 수 있으며, 삭제하면 설정 복원이나 저장 목록이 사라질 수 있습니다.

광고와 쿠키

AdSense가 포함된 문서 페이지에서는 Google이 광고 제공, 빈도 제어와 부정 이용 방지를 위해 쿠키 또는 유사 기술을 사용할 수 있습니다. 광고 맞춤설정은 Google 광고 설정과 브라우저의 쿠키 설정에서 관리할 수 있습니다.

외부 처리자

기능에 따라 Google Analytics, Google AdSense, Firebase와 Frankfurter API가 사용됩니다. 각 서비스가 처리하는 정보와 보관 방식에는 해당 제공자의 정책도 적용됩니다. 문의 이메일은 이메일 서비스 제공자의 시스템을 거칩니다.

보관과 삭제

브라우저 저장값은 사용자가 사이트 데이터나 항목을 삭제할 때까지 기기에 남을 수 있습니다. 서버에 저장되는 신청·계정·문의 기록은 기능 제공, 분쟁 대응과 법적 의무에 필요한 기간 동안 보관한 뒤 삭제하거나 식별하기 어렵게 처리합니다.

이용자의 선택
