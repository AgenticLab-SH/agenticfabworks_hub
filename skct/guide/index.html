사용 가이드

처음 쓰는 사람이 헷갈리지 않는 순서

SKCT 연습 도구는 문제를 제공하는 사이트가 아니라, 사용자가 가진 문제를 실제 온라인 검사 화면처럼 풀고 기록하는 보조 도구입니다. 아래 순서대로 사용하면 됩니다.

1. 타이머 시작

기본 세팅은 과목 시간과 쉬는 시간이 자동으로 이어지도록 잡혀 있습니다. 먼저 타이머를 시작하고 문제 풀이를 진행합니다.

2. OMR 마킹

문제를 풀면서 OMR에 답을 표시합니다. 일반 모드는 실제 시험 흐름처럼 이전 문항 수정이 제한될 수 있고, 고급 자유 풀이에서는 더 유연하게 활용할 수 있습니다.

3. 정답 입력

풀이가 끝나면 정답 입력 버튼을 눌러 정답을 입력합니다. CBT 결과 표를 복사해 붙여넣는 방식도 고급 모드에서 지원합니다.

4. 채점과 복기

채점 후 응답 수, 미응답 수, 정답 수, 정답률을 확인하고 과목별 상세 통계로 약한 문항과 오래 걸린 문항을 정리합니다.

광고와 집중 화면

일반 안내 페이지에는 광고가 표시될 수 있지만, 고급 모드와 팝업 풀이창처럼 집중해서 조작하는 화면에는 광고를 넣지 않는 방향으로 운영합니다.
