기본 기능은 모두 무료입니다. 아래 네 단계만 맞추면 바로 연습할 수 있습니다.

연습 도구 열기상세 가이드

문제 자료 열기

가지고 있는 PDF, E-book이나 CBT 자료를 평소 사용하는 화면에 엽니다.

연습 도구를 팝업으로 열기

PC에서는 팝업으로 열어 문제 자료 옆에 두는 방식을 권장합니다.

시간 맞추기

자료의 전체 시간과 과목 시간을 설정한 뒤 타이머를 시작합니다.

필요한 도구 사용하기

계산기, 메모장과 그림판을 필요한 순간에 열어 화면 조작과 시간 관리를 함께 익힙니다.

무료로 제공되는 기능

타이머, 화면 계산기, 메모장, 그림판과 개인 자료용 기본 채점은 이용권 없이 쓸 수 있습니다.

고급 모드가 필요한 경우

여러 회차를 저장하고 문항 시간, 오답 이유와 성장 흐름을 비교하고 싶을 때 선택하면 됩니다.
