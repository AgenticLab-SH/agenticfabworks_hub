자료는 그대로 열고,
도구만 옆에 두세요.

기본 기능은 가입 없이 무료로 사용할 수 있습니다. 아래 순서대로 한 번 실행하면 다음 회차부터는 저장된 화면 설정을 확인하고 바로 시작할 수 있습니다.

연습 도구 열기화면 점검표
예상 준비 시간: 2분권장 환경: PC 최신 브라우저문제 자료 업로드 불필요

문제 자료를 별도 창으로 엽니다

보유한 PDF, E-book이나 CBT 자료를 평소 사용하는 뷰어에서 엽니다. 저작권이 있는 문제 원문은 이 사이트에 올리거나 공개 커뮤니티에 공유하지 않습니다.

‘팝업으로 열기’를 눌러 옆에 배치합니다

팝업이 차단되면 주소창의 차단 표시에서 이 사이트를 허용합니다. 문제 창을 약 70%, 도구 창을 약 30%로 두고 글씨와 계산기 버튼이 편하게 보이도록 조정합니다.

전체 시간과 과목 시간을 확인합니다

이전 설정이 남아 있을 수 있으므로 숫자를 눈으로 확인합니다. 브라우저 배율과 창 너비는 한 회차 동안 고정해야 문항 시간 비교가 덜 흔들립니다.

이번 회차에서 볼 행동 하나를 정합니다

예: 90초 넘은 문항 표시, 계산 전 기준값 메모, 종료 2분 전 검토 시작. 정답률과 별개로 확인할 수 있는 행동이 좋습니다.

풀이 중에는 필요한 도구만 엽니다

계산기, 메모장과 그림판을 번갈아 사용합니다. 자세한 회고 대신 ‘조건 누락’, ‘입력 수정’, ‘시작 지연’처럼 짧은 신호만 남깁니다.

끝난 뒤에만 답안과 원인을 정리합니다

채점이 필요할 때 내 답과 정답을 입력합니다. 오래 걸린 정답과 빠른 오답을 따로 보고 가장 반복된 원인 하나를 다음 회차 행동으로 옮깁니다.

무료 기본 기능

타이머, 화면 계산기, 메모장, 그림판과 개인 자료용 기본 채점은 이용권 없이 사용할 수 있습니다. 기본 연습을 위해 가입할 필요가 없습니다.

고급 모드가 필요한 경우

여러 회차를 계정에 저장하고 문항 시간, 과목별 통계, 오답 원인과 성장 흐름을 비교하려는 경우 선택할 수 있습니다. 필수 연습 기능은 아닙니다.

기록을 남기지 않아도 되는 경우

화면 조작이나 타이머 흐름만 짧게 익히려면 답안 기록을 건너뛰어도 됩니다. 기록 목적이 없는 회차에 입력 단계를 억지로 추가하지 않습니다.

더 자세한 기능 설명

실제 버튼 위치와 설정 화면은 연습 도구 안의 ‘사용 안내’에서 확인할 수 있습니다. 이 페이지는 한 회차를 운영하는 순서와 판단 기준에 집중합니다.
