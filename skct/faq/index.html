기본 기능은 무료인가요?

타이머, 화면 계산기, 메모장, 그림판과 개인 자료용 기본 채점은 가입과 이용권 없이 사용할 수 있습니다. 여러 회차를 계정에 저장하고 문항별 시간과 성장 흐름을 비교하는 기능만 선택형 고급 모드로 분리되어 있습니다.

문제나 기출문제를 제공하나요?

제공하지 않습니다. 사용자가 합법적으로 보유한 PDF, E-book이나 CBT 자료를 별도 창에 열고 그 옆에서 쓰는 화면 도구입니다. 문제 원문과 정답 데이터는 사이트에 업로드할 필요가 없습니다.

실제 시험에도 OMR이 있나요?

이 사이트의 OMR 형태 답안 기록은 개인 자료를 채점하고 복기하기 위한 선택 기능입니다. 실제 온라인 시험 화면이나 답안 입력 방식을 그대로 재현한다는 뜻이 아닙니다.

공식 SKCT 서비스인가요?

아닙니다. SK그룹 또는 SKCT 공식 운영기관과 제휴되지 않은 개인 제작 연습 도구입니다. 실제 시험 구성, 응시 규정과 일정은 공식 채용 안내를 확인해야 합니다.

팝업이 열리지 않으면 어떻게 하나요?

주소창 오른쪽의 팝업 차단 표시에서 이 사이트의 팝업을 허용한 뒤 ‘팝업으로 열기’를 다시 누릅니다. 원래 탭과 팝업을 동시에 조작하지 말고 실제 연습에는 한 창만 사용하세요.

문제 창과 도구 창 비율은 어떻게 정하나요?

처음에는 문제 자료 약 70%, 도구 약 30%에서 시작한 뒤 표가 잘리지 않고 계산기 숫자가 편하게 보이도록 조정합니다. 한 회차가 시작된 뒤에는 창 너비와 브라우저 배율을 유지하는 편이 시간 비교에 좋습니다.

모바일에서도 사용할 수 있나요?

페이지를 열고 설정이나 기록을 확인할 수는 있지만 문제 자료, 계산기, 메모와 타이머를 동시에 보기 어렵습니다. 실제 시간 측정 연습은 PC 또는 넓은 태블릿 화면을 권장합니다.

답안 기록을 꼭 사용해야 하나요?

아닙니다. 화면 배치와 시간 흐름만 연습하려면 답안 기록을 건너뛰어도 됩니다. 채점과 복기가 필요한 회차에만 내 답과 정답을 입력하세요.

정답률이 실제 점수나 합격 가능성을 뜻하나요?

아닙니다. 기록은 사용자가 입력한 답과 정답을 기준으로 한 개인 연습 결과입니다. 실제 시험 난이도, 표준화 점수와 합격선을 반영하거나 예측하지 않습니다.

고급 모드는 무엇이 다른가요?

회차 기록, 문항별 시간, 과목별 통계, 오답 원인, 성장 그래프와 복기 메모를 계정 기준으로 저장하고 비교할 수 있습니다. 기본 타이머와 화면 도구를 쓰는 데는 필요하지 않습니다.

브라우저에 무엇이 저장되나요?

일반 모드의 화면 설정과 타이머 기본값 일부는 해당 브라우저 저장소에 남을 수 있습니다. 브라우저 데이터를 지우거나 다른 기기로 옮기면 유지되지 않을 수 있습니다.

고급 기록은 어디에 저장되나요?

로그인 기반 보관함 기록은 계정 식별자와 함께 서버에 저장됩니다. 처리 목적, 보관과 삭제 요청 방법은 개인정보 처리 안내에서 확인할 수 있습니다.

풀이 화면에도 광고가 나오나요?

타이머, OMR, 팝업과 개인 기록처럼 집중해서 조작하는 화면에는 광고 로더를 두지 않습니다. 광고 코드는 읽을 콘텐츠가 있는 소개·가이드·FAQ 페이지에만 제한합니다.

커뮤니티에 문제 사진을 올려도 되나요?

직접 만든 예시가 아니라면 저작권이 있는 문제 원문, 기출로 오해될 수 있는 화면과 개인정보가 포함된 이미지를 올리지 마세요. 오류 제보는 문제 내용이 보이지 않게 가리고 UI 부분만 첨부하는 것이 안전합니다.

오류를 제보할 때 무엇을 적나요?

발생한 URL, 운영체제와 브라우저, 화면 크기, 누른 순서와 재현 여부를 적어주세요. 저장 문제라면 새 브라우저에서도 같은지, 팝업 문제라면 차단 표시가 있었는지도 함께 알려주면 확인이 빠릅니다.

기능 제안과 개인정보 삭제 요청은 어디로 보내나요?

AgenticFabWorks 문의 페이지의 이메일로 보낼 수 있습니다. 개인정보 요청은 어떤 서비스와 계정인지 식별 가능한 최소 정보만 적고 비밀번호나 인증 코드를 보내지 마세요.
