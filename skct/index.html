Who it helps

이런 불편을 줄이기 위해 만들었습니다.

문제를 대신 제공하거나 점수를 예측하는 서비스가 아니라, 사용자가 가진 자료로 연습할 때 필요한 화면 조작과 기록을 정리하는 도구입니다.

창을 계속 바꾸는 경우

문제 PDF, 기본 계산기, 메모장과 타이머를 각각 열어 전환하느라 흐름이 끊긴다면 한 팝업 안에 필요한 도구를 모아둘 수 있습니다.

시간이 어디서 늘었는지 모르는 경우

정답 여부와 문항별 시간을 함께 남겨 오래 걸린 정답, 빠르게 판단한 오답과 건너뛴 문항을 나눠 볼 수 있습니다.

복기 문장이 매번 같은 경우

‘계산 실수’에서 끝내지 않고 조건 누락, 단위 혼동, 입력 수정처럼 원인을 나눠 다음 회차의 한 가지 행동 목표로 연결합니다.

실제 자료로 화면 연습을 하려는 경우

사용자가 합법적으로 보유한 PDF, E-book이나 CBT 자료를 그대로 사용합니다. 자료를 이 사이트에 업로드할 필요가 없습니다.

One session

한 회차는 다섯 단계로 운영합니다.

자료와 화면을 준비합니다

문제 자료를 먼저 열고 연습 도구 팝업을 옆에 둡니다. 브라우저 배율과 창 너비는 회차 중 바꾸지 않습니다.

관찰할 행동 하나를 정합니다

예: 90초 이상 걸린 문항 표시, 비율 계산 전 기준값 메모, 종료 2분 전 검토 시작.

타이머와 필요한 도구만 사용합니다

풀이 중에는 자세한 회고를 쓰지 않고 시간 지연, 조건 누락, 계산 입력 수정처럼 짧은 신호만 남깁니다.

답안과 시간을 함께 봅니다

정답률만 보지 않고 오래 걸린 정답, 빠른 오답, 건너뜀과 시간 종료 미입력을 나눠 확인합니다.

다음 회차 행동을 한 줄로 적습니다

가장 반복된 원인 하나를 골라 화면에서 확인 가능한 행동으로 바꾸고 다음 연습의 목표로 사용합니다.

제공하지 않는 것

공식 문제, 기출문제, 문제 원문과 정답 데이터는 제공하지 않습니다. 기록은 사용자가 직접 입력한 값을 기준으로 계산합니다.

도구가 예측하지 않는 것

연습 기록으로 실제 점수, 합격 가능성이나 시험 난이도를 예측하지 않습니다. 시험 구성과 규정은 공식 채용 안내를 확인해야 합니다.

비공식 서비스

SK그룹 또는 SKCT 공식 운영기관과 제휴된 서비스가 아닙니다. 브랜드와 시험 명칭은 연습 목적을 설명하기 위해서만 사용합니다.

광고 배치 기준

읽을 콘텐츠가 있는 안내 페이지에만 광고 로더를 두고, 타이머·OMR·팝업과 개인 기록처럼 집중해서 조작하는 화면에는 광고를 넣지 않습니다.
