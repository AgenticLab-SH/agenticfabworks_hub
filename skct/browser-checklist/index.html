SKCT Practice Note 02

타이머를 누르기 전에,
화면부터 고정하세요.

연습 도중 팝업이 막히거나 문제 자료가 가려지면 시간 기록이 화면 문제와 섞입니다. 처음 2분 동안 환경을 확인하면 이후 회차를 같은 조건으로 비교하기 쉬워집니다.

운영·작성: AgenticFabWorks게시: 2026년 7월 23일Windows·macOS의 최신 Chrome 계열 기준

1. 문제 자료를 먼저 엽니다

사용자가 보유한 PDF, E-book 또는 CBT 자료를 평소 쓰는 뷰어에서 엽니다. SKCT 도구에는 문제 원문을 업로드할 필요가 없습니다. 문제 자료와 연습 도구가 서로 다른 창에 있어야 창 크기를 독립적으로 조절할 수 있습니다.

  • PDF 페이지가 한 번에 너무 작게 보이면 문제 창을 먼저 넓힙니다.
  • 자료 뷰어의 페이지 이동 단축키가 입력창과 충돌하지 않는지 확인합니다.
  • 저작권이 있는 자료는 공개 커뮤니티나 피드백 첨부에 올리지 않습니다.

2. 팝업 허용은 실제 버튼으로 확인합니다

주소창 설정만 보고 끝내지 말고 연습 도구의 ‘팝업으로 열기’를 한 번 눌러 별도 창이 열리는지 확인합니다. 브라우저가 팝업을 막으면 주소창 오른쪽의 차단 표시에서 이 사이트의 팝업을 허용한 뒤 다시 시도합니다.

팝업 창이 열렸다면 원래 탭과 팝업 중 하나만 실제 연습에 사용합니다. 같은 타이머를 두 창에서 동시에 조작하면 저장 상태와 표시 시간이 다르게 보일 수 있습니다.

3. 문제 70%, 도구 30%에서 시작합니다

정답 비율은 없습니다. 다만 처음에는 문제 자료가 화면의 약 70%, 연습 도구가 약 30%를 차지하게 둔 뒤 계산기 숫자와 메모 글씨가 편하게 보이는지 조정하는 방법이 빠릅니다.

증상먼저 바꿀 것이유
문제 표가 잘려 가로 스크롤이 생김문제 창 너비 확대내용을 여러 번 이동하면 풀이 시간과 화면 조작 시간이 섞입니다.
계산기 숫자를 잘못 누름도구 창 너비 또는 브라우저 배율 확대입력 수정 횟수를 줄이는 것이 우선입니다.
메모가 문제를 가림두 창을 겹치지 않게 좌우 고정창을 계속 앞으로 가져오는 동작을 없앨 수 있습니다.

4. 브라우저 배율은 회차 중 바꾸지 않습니다

배율을 바꾸면 버튼 크기와 줄바꿈, 스크롤 위치가 달라집니다. 연습 시작 전 90%, 100%, 110% 중 읽기 편한 값을 고르고 한 회차 동안 유지합니다. 다음 회차와 비교할 때도 가능하면 같은 값을 사용합니다.

운영체제의 디스플레이 배율과 브라우저 배율은 별개입니다. 다른 PC에서 기록 차이가 커졌다면 제한 시간보다 먼저 화면 배율과 창 너비가 같은지 확인합니다.

5. 30초 시험 실행으로 저장과 소리를 확인합니다

전체 시간과 과목 시간을 넣고 30초만 실행해 일시정지, 과목 전환, 초기화가 예상대로 동작하는지 확인합니다. 소리를 사용하는 경우 기기 볼륨이 지나치게 크지 않은지도 이때 확인합니다.

이전에 저장한 설정이 남아 있다면 현재 자료의 제한 시간과 다른 값이 불러와질 수 있습니다. 설정 화면의 숫자를 눈으로 확인하고 시작해야 오래된 기본값을 그대로 사용하는 실수를 막을 수 있습니다.

시작 직전 20초 점검

문제 첫 페이지 · 팝업 한 개 · 배율 고정 · 전체/과목 시간 · 답안 기록 필요 여부. 다섯 항목이 맞으면 타이머를 시작합니다.

모바일은 짧은 확인용으로 봅니다

스마트폰에서도 페이지를 열 수 있지만 문제 자료, 계산기, 메모와 타이머를 한 화면에 함께 두기 어렵습니다. 모바일에서는 설정 확인이나 짧은 기록 검토에 쓰고, 실제 시간 측정 연습은 PC 또는 넓은 태블릿 화면을 권장합니다.
