About AgenticFabWorks

직접 써 보고,
불편한 부분부터 고칩니다.

AgenticFabWorks는 취업 준비와 일상에서 반복되는 작은 작업을 줄이기 위해 운영하는 1인 웹 도구 프로젝트입니다. 기능 수를 늘리는 것보다 사용자가 지금 하려는 일을 끝낼 수 있는지를 먼저 봅니다.

운영·작성: AgenticFabWorks최근 갱신: 2026년 7월 23일광고 없는 운영 정보 페이지

시작한 이유

SKCT 문제 자료를 보면서 타이머, 계산기, 메모장과 그림판을 창마다 따로 여는 과정이 번거로웠습니다. 이 도구들을 한 화면에 묶고 팝업 배치와 기록 흐름을 반복해서 조정한 것이 SKCT 온라인 연습 도구의 시작입니다.

직접 확인하는 범위

PC와 모바일의 주요 화면 너비, 팝업 전환, 타이머와 답안 기록, 브라우저 저장, 로그인 기반 보관함을 실제 동작 기준으로 확인합니다. 바뀐 기능은 사용 안내와 FAQ에도 함께 반영합니다.

제공하지 않는 것

공식 SKCT 문제, 기출문제, 정답 데이터와 저작권이 있는 문제 원문을 수집하거나 제공하지 않습니다. 사용자가 합법적으로 보유한 자료 옆에서 쓰는 화면 도구만 제공합니다.

공식 관계

SK그룹 또는 SKCT 공식 운영기관과 제휴된 서비스가 아닙니다. 시험 구성, 응시 규정과 최신 공지는 반드시 공식 채용 안내를 기준으로 확인해야 합니다.

How we work

기능 하나를 공개하기까지

설명용 기능을 먼저 만들지 않고, 실제 사용 흐름에서 필요한지부터 확인합니다.

불편을 한 문장으로 적습니다

예를 들어 “문제 자료와 계산기 사이를 오가느라 풀이 흐름이 끊긴다”처럼 사용 상황과 손실을 함께 적습니다.

가장 짧은 해결 흐름을 만듭니다

버튼 수보다 시작부터 종료까지 필요한 동작 수를 줄입니다. 기본 기능은 가입 없이 바로 쓸 수 있게 두는 것이 원칙입니다.

여러 화면에서 직접 검증합니다

데스크톱, 태블릿과 모바일 너비에서 겹침과 가로 넘침을 확인하고, 새 브라우저와 저장값이 남은 브라우저를 나눠 점검합니다.

한계를 문서에 남깁니다

실제 시험과 다른 부분, 모바일에서 불편한 부분, 저장 위치와 개인정보 처리 범위를 사용 안내에 명시합니다.

Editorial boundary

콘텐츠 작성 기준

도구 사용 경험을 설명합니다

가이드에는 화면을 어디에 두는지, 어떤 값을 먼저 기록하는지, 기록을 다음 연습에 어떻게 연결하는지처럼 직접 실행할 수 있는 절차를 씁니다. 검색어를 늘리기 위한 일반론은 별도 페이지로 만들지 않습니다.

보장할 수 없는 결과를 약속하지 않습니다

연습 기록은 학습 상태를 관찰하는 보조 자료일 뿐, 합격 가능성이나 실제 점수를 예측하지 않습니다. 계산 도구의 결과도 중요한 결정 전에는 공식 기준과 원자료를 다시 확인해야 합니다.
